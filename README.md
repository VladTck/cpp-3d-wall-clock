# 3D Pendulum Clock

This project is a simple interactive 3D simulation of a pendulum wall clock created using C++ and OpenGL.  
It was developed as part of a computer graphics course at university.

The application displays a 3D clock model with moving elements such as clock hands, a pendulum, and internal gears.  
The user can interact with the scene by moving the camera, rotating the object, and enabling or disabling different visual elements.

---

## Technologies

- C++
- OpenGL
- GLFW
- STB Image (for textures)
- Visual Studio 2022

---

## Project Features

- 3D clock body with wood texture
- Clock dial with hour markers
- Moving hour, minute and second hands
- Pendulum animation
- Rotating internal gears
- Scene lighting
- Camera movement and rotation
- Ability to show or hide different elements in the scene
- Perspective and orthographic view modes

---

## Controls

| Key | Action |
|----|------|
| = | Increase clock speed |
| - | Decrease clock speed |
| ESC | Exit application |
| F2 | Show / hide internal mechanism |
| F3 | Turn lighting on / off |
| F4 | Show / hide reference object |
| W | Move camera closer |
| S | Move camera away |
| A | Rotate camera left |
| D | Rotate camera right |
| ↑ | Move camera up |
| ↓ | Move camera down |
| ← | Move clock left |
| → | Move clock right |
| E | Rotate clock clockwise |
| Q | Rotate clock counterclockwise |
| X | Move clock forward |
| Z | Move clock backward |
| I | Increase red light intensity |
| K | Decrease red light intensity |
| O | Increase green light intensity |
| L | Decrease green light intensity |
| P | Increase blue light intensity |
